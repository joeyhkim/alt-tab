# Alt Tab

A Chrome extension that switches between the two most recently viewed tabs.

## Files

- `manifest.json`: Extension metadata and shortcut wiring.
- `background.js`: Tracks tab activation history and flips to the previous tab.

## Load in Chrome

1. Open `chrome://extensions`.
2. Enable `Developer mode`.
3. Click `Load unpacked`.
4. Select this repository folder.

## Set the shortcut

1. Open `chrome://extensions/shortcuts`.
2. Find `Alt Tab`.
3. Set a shortcut for `Activate extension`.

Clicking the extension icon does the same thing as the shortcut.
